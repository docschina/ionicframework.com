开始使用 Ionic 框架

使用你所熟悉和喜爱的技术开始构建高质量的移动应用程序。

对于我们的 v1 入门指南， 请点击此处

  • 使用 CLI 编码

    使用我们的 CLI 工具来构建你的应用。

  • 用 Ionic Creator 设计

    无需编码，即可形象地构建你的应用。

  • 安装 Ionic

    首先，安装 Node.js。 然后， 在你的终端中安装最新的 Ionic 命令行工具。遵循 Android 和 iOS 平台指南安装所需的开发工具。

    如果你刚接触命令行，请阅读我们的终端教程

  • 启动一个应用

    使用我们现成的应用模板创建一个 Ionic 应用程序， 或者创建一个空白的应用模板来从头开始。查看有更有设计的市场。

    对于 v1 项目，请使用--type ionic1 flag。

    • $ ionic start myApp blank
    • $ ionic start myApp tabs
    • $ ionic start myApp sidemenu

  • 运行你的应用

    你的大部分应用都可以通过 ionic serve 在浏览器中构建。我们建议从这个工作流程开始。

    当你准备将你的应用部署到真实设备时， 请查看我们的部署指南。
