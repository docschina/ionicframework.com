Inputs

改进这篇文档 输入框对于以安全的方式收集和处理用户输入非常重要。他们应该遵循每个平台的样式和交互准则，以便他们直观地与用户交互。Ionic 使用 Angular 2 的表单库，它可以被认为是两个相关的部分，**控件**和**控件组**。 表单中的每个输入字段都有一个**控件**，一个绑定到字段中的值的函数，并执行验证。**控件组**是控件的集合。控件组处理表单提交，并提供可用于确定整个表单是否有效的高级 API。 下面列出了许多可用于设置表单样式及其各种输入字段的属性。有关表单逻辑的更多信息，请查看 [Inputs API 文档](../api/components/input/Input/)。 _有关更多信息，请查看 [API 文档](../api/components/input/Input)。_
