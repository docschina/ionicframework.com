Loading

改进这篇文档 Loading 组件是一个覆盖层，可以在指示行动时阻止用户交互。 默认情况下，它会根据模式显示一个 spinner。动态内容可以与 spinner 一起传递和显示。spinner 可以隐藏或定制为使用多个预定义的选项。即使在导航过程中，loading 指示器也显示在其他内容的顶部。 _有关更多信息，请查看 [API 文档](../api/components/loading/LoadingController)。_

基本用法

示例 示例源码

```typescript import { LoadingController } from 'ionic-angular'; export class MyPage { constructor(public loadingCtrl: LoadingController) { } presentLoading() { let loader = this.loadingCtrl.create({ content: "Please wait...", duration: 3000 }); loader.present(); } } ```
