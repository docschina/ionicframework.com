Modals

改进这篇文档 Modals可以滑入屏幕以显示一个临时的界面，通常用于注册或登录页面、信息填写页面和选项选择页面。 _若要获取更多信息，请查阅 [API 文档](../api/components/modal/ModalController)._

基础用法

示例 示例源代码

```typescript import { ModalController } from 'ionic-angular'; import { ModalPage } from './modal-page'; export class MyPage { constructor(public modalCtrl: ModalController) { } presentModal() { let modal = this.modalCtrl.create(ModalPage); modal.present(); } } ```
