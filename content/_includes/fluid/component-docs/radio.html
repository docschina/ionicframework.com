Radio

改进这篇文档 与 [checkbox](#checkbox) 类似，radio 是一个保存*布尔值*的输入组件。一般情况下，radios 与 HTML radio 输入没有区别。但是，像其他 Ionic 组件一样，radio 在每个平台上的样式都不同。与复选框不同，radio 组件组成一个组，一次只能选择一个 radio。使用 `checked` 属性设置默认值，并使用 `disabled` 属性禁用用户更改该值。 _有关更多信息，请查看 [Radio Button API 文档](../api/components/radio/RadioButton) 和 [Radio Group API 文档](../api/components/radio/RadioGroup)。_
