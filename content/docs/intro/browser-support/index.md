# 浏览器支持

Ionic 专注于通过 Cordova
来构建原生/混合应用，并增加了对
[Progressive Web Apps](/docs/resources/progressive-web-apps/) 以及 
[Electron](http://electron.atom.io) 的适配

Ionic 支持以下操作系统和浏览器：

移动端操作系统

* iOS 8+
* Windows 10 Universal App
* Android 4.4+
  * 如果将 Crosswalk 与 Cordova 配合使用，则可以支持到 4.1 版本

浏览器 

* Safari
* Chrome
* Edge
